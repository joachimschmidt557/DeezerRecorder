Spotify Web Recorder

An integrated solution for easy recording of songs from Deezer to local mp3 files.

Credits

Based on Spotify Web Recorder with the following changes:

  • Uses Deezer instead of Spotify Web Player
  • Uses the Chromium Embedded Framework instead of the outdated Gecko engine

Other libraries used:

  • nAudio: http://naudio.codeplex.com
  • CefSharp: https://github.com/cefsharp/CefSharp
  • LAME: http://lame.sourceforge.net

Is it legal?

This probably depends on where you live. For example, in Sweden, recoding songs for private use is legal.

Source

DeezerRecorder is open source software. You can view the source on GitHub.
